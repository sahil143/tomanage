# ToManage Mobile App

React Native mobile application (iOS + Android) for ToManage.

## Setup

```bash
# From root
pnpm install

# Run iOS
pnpm --filter @tomanage/mobile ios

# Run Android
pnpm --filter @tomanage/mobile android
```

## Structure

- `app/` - Expo Router screens
- `src/` - Components, services, stores
- `assets/` - Images and static assets
- `context/` - Documentation
